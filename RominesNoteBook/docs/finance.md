# 理财笔记

## 基本面分析  
收入 支出 资产 负债  
现金流 = 收入 - 支出  
净值 = 资产 - 负债  
GDP 全国生产总价值  

### 估值 
净利润 = 营业收入 - 成本  
每股收益 = 净利润 / 总股本  

#### 市盈率(PE)  
为净利润所付出的价格  
股价与每股收益的比例，PE越低意味着股价越划算  

#### 市净率(PB)  
每股股价/每股净资产  
为净资产付出的价格  

#### 市销率(PS)  
股价 / 每股销售收入  
为销售收入所付出的价格  

#### 盈利增长比率(PEG)  
为净利润及增长率所付出的价格  
增长率： 净利润的增长趋势  
PEG = 市盈率/增长率 

#### 市值  
股价 * 已发行的股份数量  

#### 其他关键字  
TTM 过去12个月，Trailing Twelve Months
