# install oracle 11gR2 on ubuntu

添加 java 环境  
```
: vim /etc/profile

......

export JAVA_HOME="/root/jdk/jdk1.8.0_221"
export PATH="$JAVA_HOME/bin:$PATH"
```

解压  
```
: apt-get install unzip
: unzip linux.x64_11g... 
```
