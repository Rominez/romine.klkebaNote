需要做的修改  
```
# 可以改成 0.0.0.0
network.host
http:port
# 节点名称
node.name: node-1

# 7.x
# 集群中的节点
discovery.seed_hosts: ["somehost"]
# 初始化时参与选主的节点
cluster.initial_master_nodes: ["nodes-1"]
```

6.x 版本的集群搭建  
```yml
cluster.name: 节点在集群中的名称
node.name:
node.master:
network.host:
http.port:
transport.tcp.port: 
discovery.zen.ping.unicast.hosts: []
discovery.zen.minimum_master_nodes:
http.cores.enabled: true
http.cores.allow-origin: "*"
```

集群状态检查 `http://host:port/_cat/health?v`
