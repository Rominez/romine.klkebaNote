通过一个bit位来表示某个元素对应的值或者状态  
key 对应元素本身  
最大占用内存 512MB，可以处理四十亿的数据  

```
$ setbit [key] [offset] [value]  
# offset 相当于数组下标，大于等于0，小于2^32=4294967296，将位图限制在 512MB  
# value 只能是0或1  

$ getbit [key] [offset]  

$ bitcount [key]
# 统计有多少value为1  

$ bitop [operation] [destkey] [key] [key] ...
# operation 可以是 and、or、not、xor  
# destkey 结果保存到这个 key
```
