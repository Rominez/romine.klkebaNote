数据以行为单位存储  
一页包含多个行  

InnoDB 的4钟行格式  
- Compact  
- Redundant  
- Dynamic  
5.7 版本默认  
- Compressed  

创建表时指定行格式  
```
$ create table ...(...) row_format=dynamic  
```

修改默认行格式  
```
$ set global innodb_default_row_format=dynamic  
```

查看表的行格式  
```
$ show table status like 'tablename'\G
```
